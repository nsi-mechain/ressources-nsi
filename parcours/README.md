1. lancer `parcours_souris.py` pour créer le parcours et le convertir en un fichier binaire compressé (+ une image png)
2. lancer `parcours2db.py` pour lire ce fichier et créer une base de données SQLite
3. `requete.sql` est un exemple de requête un peu complexe
   sur la base de données pour afficher :
   - le nombre de points du parcours
   - la distance totale parcourue
   - le nombre de virages à droite et à gauche
