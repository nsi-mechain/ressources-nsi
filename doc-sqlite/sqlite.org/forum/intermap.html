Interwiki links are hyperlink targets of the form

Tag:PageName

Such links resolve to links to PageName on a separate server identified by Tag. The Interwiki Map or "intermap" is a mapping from Tags to complete Server URLs. The current mapping is as follows:

  1. docsrc

    • Base-URL: https://sqlite.org/docsrc
    • Hash-path: /info/
    • Wiki-path: /wiki?name=

  2. fiddle

    • Base-URL: https://sqlite.org/fiddle
    • Hash-path: NULL
    • Wiki-path: NULL

  3. see

    • Base-URL: https://sqlite.org/see
    • Hash-path: NULL
    • Wiki-path: NULL

  4. src

    • Base-URL: https://sqlite.org/src
    • Hash-path: /info/
    • Wiki-path: /wiki?name=

  5. wasm

    • Base-URL: https://sqlite.org/wasm
    • Hash-path: /info/
    • Wiki-path: /wiki?name=

  6. www

    • Base-URL: https://sqlite.org
    • Hash-path: NULL
    • Wiki-path: NULL
