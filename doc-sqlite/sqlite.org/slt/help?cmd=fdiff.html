The "fdiff" command:

Usage: fossil fdiff [options] HASH1 HASH2

Compute a diff between two artifacts in a repository (either a repository identified by the "-R FILENAME" option, or the repository that contains the working directory).

All of the usual diff formatting options (--tk, --by, -c N, etc.) apply. See the "diff" command for a full list of command-line options.
