# Installation Raspberry 400 en NSI

1. Raspberry Pi OS 64 bits (Legacy) with desktop

2. Hostnames + IPs fixes `192.168.247.x`

3. désintaller *Thonny*

4. Créer environnement virtuel *Python* nommé `venv`

5. Dans un terminal :
   
   - `source ~/venv/bin/activate`
   - `pip install Jupyter jupyterlab-language-pack-fr-FR`
   - `pip install jupyterlab-language-pack-fr-FR`
   - `pip install thonny`
   - `deactivate`

6. créer deux alias `jp` et `th` dans `.bashrc`

7. Puis : `sudo apt install sqlite3 sqlitebrowser`

8. Créer comptes GitHub sur `nsi-mechain` (par le prof)

9. Configurer ssh pour *GitHub*

10. Identification git

11. Cloner
